# 猫咪咖啡馆

一个轻量、可直接在浏览器运行的中文增量游戏。你要从一家刚开张的街角猫咖起步，通过点击招待客人、扩展升级分支、完成成就与阶段事件，再通过轻量 Prestige 把店铺经营成城市里的明星猫咖。

在线游玩地址：`https://wuhenzigua.github.io/idle-cat-cafe/`

![猫咪咖啡馆界面预览占位图](./public/screenshot-placeholder.svg)

## 玩法概览

- 点击“招待一桌客人”获得 `小鱼干`
- 使用 `小鱼干` 购买升级，支持 `买 1 / 买 10 / 买最大`
- 升级带有解锁条件，按阶段逐步出现（核心、饮品、拍照打卡、活动线）
- 达成累计营收里程碑，领取 `人气` 与阶段反馈
- 每 10 点 `人气` 提供 `+5%` 全局收益
- 达成条件后解锁“明星猫咖”横幅，游戏仍可继续推进

## V1.3 新增内容

- 轻量 Prestige：达到门槛后可重开并获得永久品牌倍率（`品牌值`）
- 成就系统：围绕点击、营收、升级、离线收益等目标提供长期动力
- 阶段事件与动态文案：如午后高峰、网红探店、主题活动周
- 统计面板：总点击、总离线收益、最赚钱升级、当前每秒收益来源
- 体验优化：数字跳动反馈、键盘快捷键、移动端点击手感与分栏浏览

## 操作快捷键

- `Space` / `Enter`：招待一桌客人
- `Q` / `1`：买 1
- `W` / `2`：买 10
- `E` / `3`：买最大
- `S`：手动保存
- `M`：音效开关
- `P`：执行 Prestige（满足条件时）
